## 📋 Project Overview

This is a modern, responsive website built with Next.js and Tailwind CSS, designed specifically for fundraising conversations, pitch deck downloads, and meeting bookings. The website demonstrates Skooli's traction, operating model, and AI advantage in the African education logistics space.

## 💻 Development

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Getting Started

```bash
# Install dependencies
npm install --legacy-peer-deps

# Run development server
npm run dev

# Or use PM2 for process management
npx pm2 start ecosystem.config.js

# Build for production
npm run build
```

### Project Structure

```
src/
├── app/                 # Next.js app directory
├── components/          # Reusable components
│   ├── ui/             # Basic UI components
│   ├── layout/         # Layout components
│   └── home/           # Home page specific
├── lib/                # Utility functions
└── types/              # TypeScript definitions
```
